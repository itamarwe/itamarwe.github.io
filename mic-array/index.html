What to look for. At low frequencies the array is barely directional — the wavelength dwarfs the array, so all microphones hear almost the same signal and the response is nearly a circle (omnidirectional). As frequency rises the main lobe narrows and the array "points." Once the spacing between adjacent microphones exceeds half a wavelength (the aliasing onset), spurious grating lobes appear — the array picks up sound strongly from directions you never steered toward. A real circular array trades off radius and microphone count to push that onset above the band of interest.
